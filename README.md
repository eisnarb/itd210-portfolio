# Portfolio Website - Josh Bransford

## About this Project
The capstone project for my Web Page Design II (ITD-210) course at Northern Virginia Community College. 
Website focus is on providing information and resources regarding electronic dance music; 
what the genre is, the different subgenres, items and ways to prepare for a festival/concert, where to find concerts, etc.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, RWD)
- JavaScript

## Course
ITD 210 Web Development - Northern Virginia Community College
